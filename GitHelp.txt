Комманды для регистрации данных
git config --global user.name "qasecap"
git config --global user.email "qasecap@gmail.com"

Команды только для одного проекта
git init
git remote add origin https://github.com/qasecap/oop-62-2.git

Основные команды для работы с Git
git status | состояние проекта
git add .  | скопировать все в буфер обмена
git commit -m "lesson1" | ссылка на сохранение
git push origin master
git branch
git commit -m "update githelp.txt"

git branch branch_name | создание ветки
git switch branch_name | задание имени ветки
git checkout branch_name | переключение на ветку

git switch -c branch_name
git checkout -b branch_name
